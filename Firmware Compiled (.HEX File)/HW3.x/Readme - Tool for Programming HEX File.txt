1. Download flasher tool here: https://github.com/afch/GA_Flasher/archive/master.zip
2. Run "GA_Flasher.exe" from Win32 or Win64 folder.
3. Select the port to which the device is connected.
4. Select .HEX file with firmware.
5. Select Device = Arduino Mega 2560 (see "Select Proper Arduino(Mega 2560).png" file)
6. Press "FLASH!" button and wait until a "Thank you" message appears. (see "Thank you Console Screenshot.png")
